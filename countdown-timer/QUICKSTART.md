# 快速开始指南

## 🚀 立即开始使用

### 方法1: 直接打开（最简单）
1. 找到 `index.html` 文件
2. 双击打开，即可在浏览器中使用
3. 无需安装任何软件

### 方法2: 本地服务器（推荐）
1. 在终端中运行：
```bash
cd /private/tmp/countdown-timer
python3 -m http.server 8000
```
2. 在浏览器中访问：`http://localhost:8000`

## 📱 使用步骤

### 1️⃣ 使用预设场景
- 点击任意预设场景卡片（如"煮面 20分钟"）
- 倒计时器会自动创建并开始计时
- 你可以创建多个不同场景的倒计时

### 2️⃣ 创建自定义倒计时
- 点击右上角的"+ 添加倒计时"按钮
- 输入倒计时名称（如"工作时间"）
- 设置分钟和秒数
- 点击"创建倒计时"

### 3️⃣ 控制倒计时
- **开始**: 点击"开始"按钮开始计时
- **暂停**: 暂停当前计时
- **重置**: 重置到初始时间
- **删除**: 删除当前倒计时器

## 💡 使用技巧

### 多任务管理
- 同时创建多个倒计时器管理不同任务
- 每个倒计时器独立运行，互不干扰

### 番茄工作法
- 使用"番茄钟"预设（25分钟）
- 工作25分钟，休息5分钟
- 循环使用提高效率

### 会议管理
- 会议前设置倒计时提醒
- 避免会议超时
- 轻松管理会议时间

### 运动健身
- 热身运动5分钟
- 力量训练30分钟
- 有氧运动15分钟

## 🎯 常见问题

**Q: 如何停止所有倒计时？**
A: 每个倒计时器需要单独暂停或删除

**Q: 倒计时结束后会怎样？**
A: 会播放提示音，倒计时器会显示脉冲动画效果

**Q: 可以修改预设场景吗？**
A: 可以使用预设创建新倒计时，也可以完全自定义

**Q: 支持移动设备吗？**
A: 是的，网页是响应式的，支持手机和平板使用

## 🎨 自定义设置

### 修改预设场景
编辑 `presets.js` 文件，添加你需要的场景

### 调整计时时间
通过"添加倒计时"功能自定义任意时间

### 更改主题颜色
修改 `styles.css` 中的颜色变量
