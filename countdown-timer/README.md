# 炫酷倒计时器

一个功能丰富的倒计时网页应用，支持多个倒计时器和预设场景。

## 功能特点

### ✨ 炫酷动画效果
- 霓虹发光效果
- 流动背景粒子
- 进度环动画
- 倒计时结束脉冲效果
- 平滑过渡动画

### 🎯 核心功能
- **基础倒计时**: 自定义输入分钟和秒数
- **多个倒计时器**: 同时运行多个独立的倒计时器
- **预设场景**: 快速选择常见场景的时间设置

### 🎨 预设场景
- **烹饪**: 煮面(20分钟)、炖菜(30分钟)、煎蛋(15分钟)等
- **会议**: 短会议(10分钟)、常规会议(30分钟)、长会议(60分钟)等
- **学习**: 番茄钟(25分钟)、深度学习(45分钟)等
- **运动**: 热身运动(5分钟)、有氧运动(15分钟)等
- **冥想**: 5-45分钟的各种冥想时间

## 安装使用

### 方法1: 直接打开
1. 将所有文件放在同一目录中
2. 用浏览器打开 `index.html` 文件
3. 即可使用

### 方法2: 本地服务器
```bash
# 使用Python 3
python -m http.server 8000

# 使用Node.js http-server
npx http-server

# 使用PHP
php -S localhost:8000
```
然后在浏览器中访问 `http://localhost:8000`

## 快速开始

### 使用预设场景
1. 在"预设场景"区域点击任意场景卡片
2. 倒计时器将自动创建并开始计时

### 创建自定义倒计时
1. 点击"+ 添加倒计时"按钮
2. 输入倒计时名称
3. 设置分钟和秒数
4. 点击"创建倒计时"

### 控制倒计时器
- **开始**: 开始计时
- **暂停**: 暂停当前计时
- **重置**: 重置倒计时到初始状态
- **删除**: 删除当前倒计时器

## 文件说明

- `index.html`: 主页面，包含所有结构和布局
- `styles.css`: 样式文件，包含所有动画和视觉效果
- `script.js`: JavaScript逻辑，包含倒计时管理器
- `presets.js`: 预设场景配置

## 自定义扩展

### 添加更多预设场景
编辑 `presets.js` 文件，在相应的数组中添加新场景：

```javascript
const newScene = {
    minutes: 10,
    name: '新场景',
    icon: '🎯'
};
presets.category.push(newScene);
```

### 修改动画效果
编辑 `styles.css` 文件中的 `@keyframes` 定义，可以自定义动画效果。
